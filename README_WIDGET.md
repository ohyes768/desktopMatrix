# DesktopWidget - 桌面四象限小工具

## 🎯 项目简介

DesktopWidget 是一个极简的桌面四象限任务管理小工具，灵感来自高效能人士的时间管理四象限方法。工具固定在桌面角落，占用空间极小，让你随时查看和管理重要任务。

### ✨ 核心特点

- 📱 **极小尺寸** - 280x320像素的紧凑设计
- 🎯 **四象限管理** - 重要紧急、重要不紧急、紧急不重要、不重要不紧急
- 💻 **桌面集成** - 固定在桌面角落，不干扰工作
- ⚡ **快速操作** - 一键添加任务，拖拽分类，右键管理
- 💾 **数据安全** - 本地SQLite存储，数据永不丢失

## 🎮 使用方法

### 基本操作
1. **添加任务** - 点击"+"按钮或按快捷键 Ctrl+T
2. **完成任务** - 勾选任务左侧复选框
3. **删除任务** - 右键任务选择"删除"
4. **移动任务** - 拖拽任务到不同象限
5. **隐藏窗口** - 右键标题栏选择"隐藏"

### 快捷键
- `Ctrl + T` - 添加新任务
- `Ctrl + H` - 隐藏/显示窗口
- `Ctrl + Q` - 退出应用
- `Ctrl + R` - 刷新任务列表

### 系统托盘
应用最小化到系统托盘，右键托盘图标可以：
- 显示/隐藏主窗口
- 添加快速任务
- 打开配置设置
- 退出应用程序

## 📦 安装部署

### 系统要求
- Windows 10 1903 或更高版本
- .NET 9.0 Runtime
- 至少 50MB 可用磁盘空间

### 安装步骤
1. 下载 `DesktopWidget_v1.0.exe`
2. 双击运行安装程序
3. 按提示完成安装
4. 首次运行自动创建数据文件

### 卸载步骤
1. 打开 "控制面板" > "程序和功能"
2. 找到 "DesktopWidget" 并选择卸载
3. 按提示完成卸载
4. 数据文件位于 `%APPDATA%/DesktopWidget/`，可手动删除

## 📊 数据存储

### 本地数据库
- **位置**: `%APPDATA%/DesktopWidget/data.db`
- **格式**: SQLite
- **备份**: 每日自动备份到 `%APPDATA%/DesktopWidget/backup/`

### 配置文件
- **位置**: `%APPDATA%/DesktopWidget/config.json`
- **内容**: 窗口位置、主题设置、快捷键配置

## 🔧 自定义配置

### 窗口设置
```json
{
  "position": {
    "x": 1920,
    "y": 20
  },
  "size": {
    "width": 280,
    "height": 320
  },
  "opacity": 85,
  "alwaysOnTop": true
```

## 🐛 故障排除

### 常见问题

**Q: 应用无法启动**
A: 检查是否安装了 .NET 9.0 Runtime，尝试以管理员身份运行

**Q: 窗口显示异常**
A: 重置窗口位置：删除 `%APPDATA%/DesktopWidget/config.json` 后重启

**Q: 数据丢失**
A: 检查备份文件 `%APPDATA%/DesktopWidget/backup/`，可手动恢复

**Q: 快捷键不工作**
A: 检查是否有其他程序占用了相同快捷键，可在设置中修改

### 日志文件
- **位置**: `%APPDATA%/DesktopWidget/logs/`
- **格式**: 按日期分割的文本文件
- **用途**: 记录应用运行错误和调试信息
